# Tree-sitter parser for Groovy

## Features
- supports most groovy features, including:
   - classes
   - control flow
   - string interpolation
   - closures
   - imports
- tree-sitter queries for
  - highlights
  - indents
  - locals
- rich parse tree to support other extensions like TreeSJ, textobjs (WIP)
